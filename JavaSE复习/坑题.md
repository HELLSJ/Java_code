## 类和对象
### 第一题：打印的内容
```java
class Base {
    public Base(String s) {
        System.out.print("B");
    }
}

class Derived extends Base {
    public Derived(String s) {
        super(s);
        System.out.print("D");
    }

    public static void main(String[] args) {
        new Derived("C");
    }
}

```
打印B和D，不打印C！

执行：

`new Derived("C");`

等价于：

`new Derived("C") → 调用 Derived(String s)`

第一步：进入子类构造方法

```java
public Derived(String s) {     
	super(s);       
	... 
}
```

这里的 `s` 接收到 `"C"`，但 **没有打印**。

第二步：进入父类构造方法

`public Base(String s) {     System.out.print("B");  // 打印 B }`

仍然只有 `"B"`，并没有打印 s。

第三步：回到子类构造方法

`System.out.print("D");   // 打印 D`

也不打印 `"C"`。

### 第二题：打印的内容

```java
class Base {
    Base() {
        System.out.print("Base");
    }
}

class Alpha extends Base {
    public static void main(String[] args) {
        new Alpha();   // 输出 “Base”
        new Base();
    }
}

```

解释：
- Alpha 没写构造方法
- Java 默认添加一个无参构造：

```java
public Alpha() {
    super();  // 自动添加的！
}
```

执行 new Alpha() 时：

`先调用父类 Base() 输出 "Base"`

所以结果正确。

### 第三题：输出

```java
class X {    
    Y y = new Y(); // 第 1 步：成员变量初始化
    public X() {   // 第 2 步：构造方法
        System.out.print("X");
    }
}

class Y {
    public Y() {
        System.out.print("Y");
    }
}

class Z extends X {
    Y y = new Y(); // 第 4 步：子类成员变量初始化
    public Z() {   // 第 5 步：子类构造
        System.out.print("Z");
    }
}

public static void main(String[] args) {
    new Z();
}
```
